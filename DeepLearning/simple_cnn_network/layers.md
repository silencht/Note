### 13 [copy()](https://www.runoob.com/python3/python3-att-list-copy.html)

copy()和直接=赋值的区别：

```python
a=[0,1,2,3,4,5]
b=a
c=a.copy()

del a[1]  
'''
   各变量值为：
   a=[0, 2, 3, 4, 5]
   b=[0, 2, 3, 4, 5]
   c=[0, 1, 2, 3, 4, 5]
'''

b.remove(4)
'''
   各变量值为：
   a=[0, 2, 3, 5]
   b=[0, 2, 3, 5]
   c=[0, 1, 2, 3, 4, 5]
'''

c.append(9)
'''
   各变量值为：
   a=[0, 2, 3, 5]
   b=[0, 2, 3, 5]
   c=[0, 1, 2, 3, 4, 5, 9]
'''
```

可以看出，使用=直接赋值，是引用赋值，更改一个，另一个同样会变, **例子中的a,b改变两次都影响到了对方**

copy() 则顾名思义，复制一个副本，原值和新复制的变量互不影响.
